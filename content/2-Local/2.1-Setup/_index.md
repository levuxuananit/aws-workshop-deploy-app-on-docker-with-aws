### Verification
By default, **Node.js** comes bundled with `npm`. To verify that **Node.js** and `npm` have been installed correctly, run the following commands in your terminal:

```bash
  node -v
  npm -v
```
If the terminal displays the version numbers (e.g., v20.x.x and 10.x.x), your installation was successful.
